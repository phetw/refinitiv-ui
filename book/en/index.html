Refinitiv's Design System Components

Element Framework provides components and tooling aligned with Refinitiv's design system to help developers build applications faster and more efficiently.

Components are flexible, cohesive and simple to use. They can work in any framework, or by themselves.

Theming is fully compliant with Refinitiv's Halo product design system

Performance is superior with render times and load speeds, using shared resources across components.
